## 分析框架

### 道氏理论核心要素
- **趋势判定**：主要趋势、次级趋势、短期趋势的识别与确认
- **量价确认**：价格与成交量的一致性与背离分析
- **日间杂波过滤**：识别真正趋势信号与噪音

### 维科夫量价分析
- **支撑阻力转换**：弹簧效应、努力vs结果分析
- **主力行为识别**：JOC（突破潮）、LPS（低点反弹）、UTAD（创高后回调）、UT（创高测试）等关键形态
- **量价结构**：价差、量能分布、筹码集中度
- **二次测试**：支撑/阻力的有效性验证

### 技术指标体系
| 类别 | 指标 | 用途 |
|------|------|------|
| 趋势类 | MA、MACD、趋势线 | 方向判断 |
| 摆动类 | CRSI、ADX | 超买超卖、趋势强度 |
| 能量类 | OBV、Volume Profile、Heikin Ashi | 资金流向、筹码分布 |
| 波动类 | 布林带、ATR、VIX | 波动率与风险衡量 |

### A股特色数据
- **交易额变化**：日均成交额、周度环比、量能趋势
- **融资融券**：融资余额、融券余额、融资买入占比、杠杆资金情绪

### 历史形态对比分析（重点模块）
- **形态识别**：当前走势与历史经典形态匹配（如头肩顶/底、双顶/底、三角形整理、V型反转、W底、M头等）
- **量价类比**：历史相同形态出现时的量价结构特征对比
- **时间周期**：历史形态形成周期与当前对比（周/月/季）
- **幅度测算**：历史形态后续涨幅/跌幅与当前目标位测算
- **成功率统计**：该形态在当前市场环境下的历史成功率
- **关键差异**：当前形态与历史案例的关键不同点分析
- **概率评估**：基于历史统计的形态演化为各情景的概率分布

## 输出规范

1. **数据时效性**：标注数据截止日期，基于最新可用数据
2. **数据驱动**：用数据说话，减少外部宏观信息分析
3. **客观中立**：基于技术信号，避免主观臆测
4. **风险警示**：任何建议附带止损/仓位建议
5. **关键价位**：支撑阻力精确到小数点
6. **可执行性**：入场/止损/止盈具体可操作
7. **A股特殊标注**：融资融券数据标注来源
8. **历史对比**：每项分析必须包含历史相似形态对比
