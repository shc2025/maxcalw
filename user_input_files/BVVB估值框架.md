# 📋 BVVB个股评分核心框架
**版本：v2.3 | 制定日期：2026-04-14 | 最新更新：2026-04-19**
**制定：如远**
**核心理念：相对比较、主观综合、打分驱动研究，不是预测工具**

---

## 一、定位与核心原则

```
打分 ≠ 交易策略
打分 = 研究全面性的检查清单 + 候选池排序依据

目的：确保研究没有盲点，对机会的判断有优先顺序
限制：分数是主观综合判断，不是精确预测，不能直接生成买卖指令
```

---

## 二、七维度评分体系（v2.2+）

| 维度 | v2.2权重 | 核心评估内容 |
|------|---------|------------|
| ①基本面 | **25%** | 质量+成长合并，分析当前已知业绩和成长性 |
| ②**估值** | **20%** | 股价上涨潜力、空间及回调风险（PE/PB/PEG/DCF三步法） |
| ③催化剂 | **25%** | 未来业绩催化剂（订单/政策/数据） |
| ④板块趋势 | **15%** | 行业趋势和板块轮动 |
| ⑤资金流向 | **5%** | 主力资金流向 |
| ⑥宏观大盘 | **5%** | 大盘和宏观经济影响 |
| ⑦股权结构 | **5%** | 股权结构和股东行为 |

> **权重总和：25%+20%+25%+15%+5%+5%+5% = 100%**

---

## 三，综合评分公式（v2.2）

```
个股综合分 =

  ①基本面   × 25%
+ ②估值     × 20%
+ ③催化剂   × 25%
+ ④板块趋势 × 15%
+ ⑤资金流向 ×  5%
+ ⑥宏观大盘 ×  5%
+ ⑦股权结构 ×  5%

= ___ / 10分
```

> **计算精度规则**
> - 每个维度独立乘以权重，贡献值保留3位小数
> - 最终综合分四舍五入至**小数点后2位**
> - 多股扫描时，所有股票必须使用同一权重和计算顺序，结果可直接横向比较

## 四、评分信号体系（v2.2）

| 分数区间 | 信号 | 操作 |
|---------|------|------|
| **8.5–10分** | 🟢 **一级强力买入** | 核心首选，重仓配置，长期持有 |
| **6.5–8.49分** | 🟢 **买入候选** | 候选池首选，等待技术面最佳时机买入 |
| **5.0–6.49分** | 🟡 观望 | 不买入，不深度研究 |
| **4.0–4.99分** | 🟠 减仓警示 | 考虑减仓，逻辑开始弱化 |
| **<4.0分** | 🔴 清仓回避 | 离场，不研究 |

**买入信号：综合分 ≥ 6.5**
**强力买入信号：综合分 ≥ 7.5 + 催化 ≥ 8.0 + 估值 ≥ 6.0**
**强制退出：综合分跌破 5.0**
**⚠️ 估值泡沫警告：综合分 ≥ 6.5 但 估值得分 ≤ 3.5**

## 五、各维度详细评分标准

### 维度①：基本面（权重25%）—— vs 全市场同类

| 分数 | 核心标准 |
|------|---------|
| 10分 | 订单环比加速 + 已公告重大新合同（>10亿）+ 量价齐升 + 毛利率提升 + ROE>20% |
| 8-9分 | 在手订单饱满（>年营收80%）+ 净利润增速>30% + ROE>18% + 壁垒持续加深 |
| 6-7分 | 订单稳定（年营收50-80%）+ 增速10-30% + ROE 12-18% |
| 4-5分 | 订单持平或略降，增速0-10%，ROE<12% |
| 2-3分 | 订单下降>20%，或核心客户流失，或出现颠覆性竞争者 |
| 0-1分 | 主营业务受根本性冲击，业绩大幅亏损无改善迹象 |

**数据必须来源（优先级）：** 公司财报 > 交易所公告 > 行业协会 > 第三方统计

### 维度②：估值（权重20%）—— v2.2+ 新增 · 三步法估值体系

> **⚠️ 不要依赖单一指标。结合 PE、PB、PEG、DCF 等多维度综合分析。**

#### 第一步：横向对比 — PE/PB 与行业均值比较

| 操作 | 方法 | 判断标准 |
|------|------|---------|
| PE横向对比 | 当前PE vs 行业中位数/均值 | 低于行业均值 → 低估加分；高于行业均值30%+ → 谨慎扣分 |
| PB横向对比 | 当前PB vs 行业PB中位数 | 破净或PB低于行业 → 资产价值被低估 |
| 跨行业对比 | 不同行业PE不可直接比较 | 消费/金融/科技估值体系不同，需在同行业内对比 |

**示例：** 若某消费类资产的市盈率为15倍，而行业均值为20倍，说明该资产在行业内处于低估状态。

#### 第二步：纵向对比 — 当前估值 vs 历史估值分位

| 操作 | 方法 | 判断标准 |
|------|------|---------|
| PE历史分位 | 当前PE所处10年/5年历史区间 | 处于历史20%分位以下 → 低估；处于80%分位以上 → 高估 |
| PB历史分位 | 当前PB所处历史区间 | 接近历史最低分位 → 极大安全边际 |
| 股价相对位置 | 当前股价 vs 52周/历史高低点 | 从高位回调50%+ → 估值风险大幅释放 |

**示例：** 某资产的市盈率当前处于历史20%分位，说明其估值低于历史上80%的时期，属于低估区间。

#### 第三步：基本面匹配 — 估值与公司质量动态匹配

| 关键问题 | 评估内容 | 低估值陷阱特征 |
|---------|---------|-------------|
| 盈利质量 | 净利润是否真实？现金流是否匹配？ | 应收账款暴增 + 经营现金流持续为负 |
| 成长持续性 | 增速来源是否可持续？ | 一次性损益、变卖资产、政府补贴撑起的低PE |
| 资产质量 | PB低是否因为资产泡沫？ | 商誉/无形资产占净资产比过高 |
| 周期位置 | 是否处于周期顶部故意压低PE？ | 强周期股在景气高点PE最低，反而是卖出信号 |
| 结构调整 | 估值压缩是否因业务模式被颠覆？ | 主业竞争力持续下滑 |

**⚠️ 关键原则：低估值可能是基本面恶化导致，而非真正低估。必须先排除低估值陷阱，再评分。**

---

#### 估值评分标准（10分制）

| 分数 | 横向（vs行业） | 纵向（vs历史） | 基本面匹配 | 综合结论 |
|------|--------------|--------------|----------|---------|
| **10分** | PE/PB均处行业最低10% | 处历史10%分位以下 | 资产扎实，盈利真实，成长未到顶 | 极度低估，安全边际极大 |
| **8-9分** | PE/PB低于行业均值30%+ | 处历史20%分位以下 | 成长持续，盈利匹配 | 低估，上升空间>30% |
| **6-7分** | PE/PB接近行业均值 | 处历史30-60%分位 | 估值合理，成长匹配 | 合理，上升空间15-30% |
| **4-5分** | PE/PB高于行业均值20%+ | 处历史60-80%分位 | 基本面尚可但偏高估 | 偏高估，上升空间<15% |
| **2-3分** | PE/PB处行业最高20% | 处历史80%分位以上 | 有一定成长但无法支撑估值 | 高估，回调风险大 |
| **0-1分** | PE>100x或无盈利；PB极高 | 创历史新高ATH | 纯属概念炒作，盈利无法证明 | 极度高估，远离估值锚 |

## 八，研究流程

```
第一步：候选股票池建立（每周扫一次）
  触发条件（任一满足即入池）：
    ✅ 基本面利好信号（订单/业绩/新品）
    ✅ 催化剂出现（政策/公告/数据）
    ✅ 板块出现资金异动
    ✅ 突发消息（24小时内评估）

第二步：对池内所有股票打分
  7个维度统一打分（10分制）
  ⚠️ 估值维度须完成三步法分析（横向→纵向→基本面匹配）
  按综合分高→低排序 → 形成池内相对排名

第三步：信号评估
  强力买入：综合分≥7.5 + 催化≥8.0 + 估值≥6.0
  买入候选：综合分 ≥ 6.5
  持有：     5.0 ≤ 综合分 < 6.5（逻辑未破坏）
  观望：     4.0 ≤ 综合分 < 5.0
  减仓：     综合分 < 4.0

第四步：操作计划
  8.5分以上：重仓买入方案（价格+仓位+止损）
  6.5-8.4分：正常买入方案
```

## 九、持仓动态管理

```
持仓股出现以下任一情况 → 立即触发重新评估：
  🚨 综合分跌破 5.0（退出信号）
  🚨 基本面核心逻辑破坏（订单丢失/业绩大幅下修）
  🚨 催化剂消失（利好未兑现/变利空）
  🚨 板块资金连续3周净流出
  🚨 估值得分由高转低（估值泡沫化）

持仓股持续满足以下条件 → 持有不动：
  ✅ 综合分维持在 ≥ 6.5
  ✅ 基本面逻辑未被破坏
  ✅ 目标价未达成
  ✅ 池内有更好的替代机会（排名高出2位以上）→ 考虑换仓
```

## 十、框架使用纪律

```
1. 不为打分而打分 → 目的是发现研究盲点，不是追求高分
2. 分数变化比绝对值重要 → 从7分降到6分要追问原因
3. 分数要每周末更新 → 市场在变，分数也要更新
4. 池子不超过20只 → 超过就主动出池，保持聚焦
5. 出现1分的维度 → 该维度对应风险必须在操作中体现
6. 每次评分必须先搜数据 → 不得凭记忆输出，数据真实性第一
7. 计算后必须用bvvb_calculator.py验算 → 结果才可发出
8. 估值评分必须完成三步法 → 横向+纵向+基本面匹配，缺一步扣分
```

## ⚠️ v2.2 → v2.3 估值部分核心更新说明

**变更主题：估值维度三步法分析体系**

| 更新项 | 原版（v2.2前） | 更新版（v2.3） |
|--------|--------------|--------------|
| 分析方法 | 单一PE/PB参考 | **三步法：横向+纵向+基本面匹配** |
| 多指标协同 | 无明确要求 | **必须结合PE、PB、PEG、DCF综合判断** |
| 低估值陷阱识别 | 无专项说明 | **新增低估值陷阱5大特征** |
| 强周期股处理 | 无区分说明 | **强周期股须在景气高点PE最低时排除** |
| 估值得分标准 | 简版表格 | **三维度综合评分表（横向+纵向+基本面）** |
| 使用纪律 | 7条 | **新增第8条：三步法缺一步扣分** |

**三步法使用规范：**
```
每只股票估值评分前，必须完成：
  ✅ 第一步：横向对比（vs行业均值）
  ✅ 第二步：纵向对比（vs自身历史分位）
  ✅ 第三步：基本面匹配（排除低估值陷阱）
  → 三个维度综合判定后，再给出估值得分（0-10分）
```
