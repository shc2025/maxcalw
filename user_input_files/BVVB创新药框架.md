# BVVB创新药框架
**版本：v1.0 | 制定日期：2026-04-24 | 制定：如远**
**定位：BVVB体系下的创新药行业专用评分框架（Specialized Version）**
**适用：PE为负/无商业化收入的创新药公司（百利天恒、恒瑞、百济神州等）**
**与主框架关系：BVVB v2.3的基础上，针对创新药行业定制化调整**

---

## 一、核心设计理念

```
创新药评分 ≠ 传统基本面评分
创新药评分 = 管线价值评估 + 催化事件驱动 + 现金消耗追踪

核心原则：
1. 管线价值优先于当期财务（当期亏损≠没有价值）
2. 催化剂权重高于一切（临床数据 = 股价）
3. 现金燃烧速度决定存续风险（ biotech最大风险：没钱撑到商业化）
4. BD交易是核心价值实现路径（license-out = 变现）
5. PE/ROE/PEG 对创新药标的系统性失效，需全面替换
```

---

## 二、八维度评分体系

| 维度 | 名称 | 权重 | 核心评估内容 |
|------|------|------|------------|
| ① | **管线进度** | **30%** | 临床阶段+III期成功率+适应症市场空间 |
| ② | **催化剂密度** | **25%** | NDA申报/获批/BD交易/临床数据读出时间线 |
| ③ | **管线广度** | **15%** | 管线数量+多适应症覆盖+梯次布局 |
| ④ | **BD价值实现** | **10%** | 已完成BD交易质量+潜在下一笔BD价值 |
| ⑤ | **现金健康度** | **10%** | 账上现金+燃烧速度+融资能力 |
| ⑥ | **竞争格局** | **5%** | 靶点独占性+me-better vs me-only |
| ⑦ | **板块趋势** | **3%** | 创新药板块资金动量 |
| ⑧ | **宏观政策** | **2%** | 医保谈判+审评审批加速+出海政策 |

> 权重总和：30%+25%+15%+10%+10%+5%+3%+2% = 100%

## 三、各维度评分标准

### 维度①：管线进度（权重30%）—— 最高权重

**核心逻辑：临床阶段越靠后，商业化概率越高，估值越扎实**

| 评分 | 临床阶段 | 核心判断 |
|------|---------|---------|
| 10分 | 已上市 + 年销售>5亿 | 商业化成熟，风险最低 |
| 9分 | NDA/BLA已受理 + 优先审评 | 已进入获批倒计时 |
| 8分 | III期临床达主要终点 | 已验证临床价值，进入报批阶段 |
| 7分 | III期进行中（期中分析积极）| 商业化概率>50%，高价值 |
| 6分 | III期进行中（中期数据待读出）| 尚不确定，进入观察 |
| 5分 | II期临床概念验证（POC）成功 | 有潜力，但III期失败率仍高 |
| 4分 | II期进行中 | 不确定性较大 |
| 3分 | I期临床进行中 | 高风险，纯早期 |
| 1-2分 | 临床前 | 极高风险，基本不考虑 |

**附加规则：**
- 多适应症同时推进 → 维度内+1分（分摊风险）
- 获得FDA突破性疗法（BTD）或孤儿药认定 → 维度内+1分
- 全球首创（First-in-class）→ 维度内+1分

---

### 维度②：催化剂密度（权重25%）—— 第二高权重

**核心逻辑：催化剂越近越确定，股价弹性越大**

| 评分 | 催化剂状态 | 时间窗口 |
|------|-----------|---------|
| 9-10分 | NDA已受理/优先审评，或BD签约在即 | 1-3个月内 |
| 7-8分 | III期主要终点达成的年内催化 | 3-6个月内 |
| 6-7分 | 重要临床数据读出（期中分析）| 6-12个月内 |
| 4-5分 | 远期催化（商业化后市场预期）| 12个月以上 |
| 1-3分 | 无明确近中期催化 | 不确定 |

**催化剂记分规则（逐项加分）：**
```
+2分：NDA/BLA已受理
+2分：FDA BTD认定（或同等效力）
+1.5分：与大型药企BD谈判进行中
+1分：重要学术会议数据公布（ASCO/ESMO等）
+1分：年内的临床数据读出
+0.5分：进入医保谈判目录
```

---

### 维度③：管线广度（权重15%）

**核心逻辑：管线越分散，抗风险能力越强，单一管线失败不至于归零**

| 评分 | 管线状态 |
|------|---------|
| 9-10分 | 3个以上III期管线 + 2个以上II期，覆盖2个以上治疗领域 |
| 7-8分 | 2个III期 + 1个以上II期 |
| 5-6分 | 1个III期 + 多个I/II期 |
| 3-4分 | 仅1个III期，其余均为早期 |
| 1-2分 | 仅I-II期，无后期管线 |

**评估要点：**
- 是否有多元化收入来源（降低单一产品失败风险）
- 管线梯次是否合理（近中长期结合）
- 治疗领域是否分散（肿瘤vs自免vs代谢）

---

### 维度④：BD价值实现（权重10%）

**核心逻辑：BD交易是创新药公司变现的主要路径，也是管线价值的市场验证**

| 评分 | BD状态 |
|------|--------|
| 9-10分 | 已完成>10亿美元级别BD + 管线仍在推进 |
| 7-8分 | 已完成>1亿美元BD，有持续现金流入 |
| 5-6分 | 曾完成BD，但近期无新进展 |
| 3-4分 | 无BD历史，正在洽谈中 |
| 1-2分 | 从未有过BD，且管线多为早期 |

**评估要点：**
- 已收首付款的BD：确认已到账资金（直接影响现金健康）
- 里程碑金：是否可触达（与临床进度挂钩）
- 未来BD潜力：靶点稀缺性+国际竞争力

---

### 维度⑤：现金健康度（权重10%）

**核心逻辑：创新药公司最大风险是"烧光现金后管线还在烧钱"**

| 评分 | 现金状态 | 评估标准 |
|------|---------|---------|
| 9-10分 | 现金充足，可支撑>3年 | 账上现金>年烧钱额的3倍 |
| 7-8分 | 现金充裕，可支撑2-3年 | 账上现金>年烧钱额的2倍 |
| 5-6分 | 现金一般，可支撑1-2年 | 账上现金约等于年烧钱额 |
| 3-4分 | 现金紧张，<12个月 | 需要融资或BD回血 |
| 1-2分 | 现金告急，<6个月 | 高风险，需要立即融资 |

**计算公式：**
```
现金可支撑时间（月）= 账上现金 ÷（月均净烧钱额）
月均净烧钱额 = （经营现金流净额绝对值 + 研发费用）÷ 12
```

## 六、管线NPV估值辅助工具（替代PE/ROE/G1）

**创新药公司不能看PE，要看管线NPV：**

```
单管线NPV估算公式：

NPV = Peak_Sales × Success_Prob × Discount / Shares

其中：
- Peak_Sales（峰值销售）：管线商业化后预期年销售峰值（人民币）
- Success_Prob（成功概率）：按临床阶段估算（I期15%，II期30%，III期55%，NDA 80%，已上市95%）
- Discount（折现系数）：管线距商业化每1年折现15%
- Shares（总股本）：调整后每股NPV

多管线：NPV_Total = Σ(NPV_i)
当前市值 vs NPV：判断溢价/折价程度
```

**判断标准：**
- 市值 < NVTotal × 0.8 → 折价（低估）
- 市值 = NPVTotal × 0.8~1.2 → 合理
- 市值 = NPVTotal × 1.2~2.0 → 溢价（可接受，看催化剂是否price-in）
- 市值 > NPVTotal × 2.0 → 严重透支（高风险区）

## 七、特殊规则

### 规则1：PE/ROE/G1 完全不适用于创新药
- 所有使用PE/G1的评分维度一律替换为"管线进度"和"现金健康度"
- 在数据表中注明：PE=N/A（EPS为负），G1=N/A（一次性BD收入不可比）

### 规则2：Biotech与Pharma的区分
```
纯Biotech（无商业化收入）：使用本框架全部8个维度
商业化Pharma（有稳定营收）：同时参考BVVB v2.3主框架
```

### 规则3：现金健康度触发式降档
```
若现金可支撑时间 < 12个月：
→ 自动触发"现金告急"警报
→ 催化剂维度-2分
→ 综合信号强制降一档（买入→观望 或 观望→减仓）
```

### 规则4：III期临床失败的处理
```
若管线III期主要终点未达成：
→ 管线进度维度重置为 1-2分
→ 综合分直接降至 <4分
→ 信号变为 🔴回避
→ 该标的从股池中移出，12个月内不再跟踪
```

### 管线NPV辅助验证

```
BL-B01D1 管线估值：
- 峰值销售：$15亿（NSCLC+鼻咽癌+食管癌，保守）
- III期成功率：55%；折现系数：0.85（距商业化约2年）
- NPV（BL-B01D1）= $15亿 × 55% × 0.85 ≈ $7.0亿 ≈ 50亿人民币
- 其他管线（SI-S9301等早期）：~5亿人民币
- 管线总NPV：~55亿人民币

当前市值：~1150亿人民币
溢价倍数：1150 ÷ 55 ≈ 21倍

⚠️ 市场已对BL-B01D1定价了非常乐观的预期（峰值销售$40-50亿？）
→ 这是高风险信号：催化剂稍有差池，股价将大幅回调
→ 建议：等下一个催化剂确认（FDA批准或新BD）再考虑介入
```
