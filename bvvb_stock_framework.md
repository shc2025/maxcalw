# 📋 BVVB个股评分核心框架
**版本：v2.0 | 2026-04-14 | 制定：如远**
**核心理念：相对比较、主观综合、打分驱动研究，不是预测工具**

---

## 一、定位与核心原则

```
打分 ≠ 交易策略
打分 = 研究全面性的检查清单 + 候选池排序依据

目的：确保研究没有盲点，对机会的判断有优先顺序
限制：分数是主观综合判断，不是精确预测，不能直接生成买卖指令
```

---

## 二、六维度评分体系

| 维度 | 权重 | 核心评估内容 | 评估基准 |
|------|------|------------|---------|
| ①基本面 | **30%** | 订单/利润/壁垒/持续性 | vs 全市场同类比较 |
| ②催化剂 | **30%** | 时间近度/确定性/大小 | vs 全市场所有股票 |
| ③板块趋势 | **10%** | 板块动量/资金/政策 | vs 所有板块排名 |
| ④资金流向 | **10%** | 主力/北向/筹码 | vs 个股自身历史 |
| ⑤宏观大盘 | **10%** | 系统性风险/流动性 | vs 历史极端区间 |
| ⑥股权结构 | **10%** | 内部人/质押/解禁 | vs 同行业水平 |

> **权重总和：30%+30%+10%+10%+10%+10% = 100%**

---

## 三、综合评分公式

```
个股综合分 =

  ①基本面   × 30%
+ ②催化剂   × 30%
+ ③板块趋势 × 10%
+ ④资金流向 × 10%
+ ⑤宏观大盘 × 10%
+ ⑥股权结构 × 10%

= ___ / 10分
```

> **计算精度规则**
> - 每个维度独立乘以权重，贡献值保留3位小数
> - 最终综合分四舍五入至**小数点后2位**
> - 多股扫描时，必须使用相同权重和计算顺序，结果可直接横向比较

## 四、评分信号体系

| 分数区间 | 信号 | 操作 |
|---------|------|------|
| **8.5–10分** | 🟢 **一级强力买入** | 核心首选，重仓配置，长期持有 |
| **6.5–8.49分** | 🟢 **买入候选** | 候选池首选，等待技术面最佳时机买入 |
| **5.0–6.49分** | 🟡 观望 | 不买入，不深度研究 |
| **4.0–4.99分** | 🟠 减仓警示 | 考虑减仓，逻辑开始弱化 |
| **<4.0分** | 🔴 清仓回避 | 离场，不研究 |

**买入信号：综合分 ≥ 6.5**
**强制退出：综合分跌破 5.0**

## 五、各维度详细评分标准

### 维度①：基本面（权重30%）—— vs 全市场同类

| 分数 | 核心标准 |
|------|---------|
| 10分 | 订单环比加速 + 已公告重大新合同（>10亿）+ 量价齐升 + 毛利率提升 + ROE>20% |
| 8-9分 | 在手订单饱满（>年营收80%）+ 净利润增速>30% + ROE>18% + 壁垒持续加深 |
| 6-7分 | 订单稳定（年营收50-80%）+ 增速10-30% + ROE 12-18% |
| 4-5分 | 订单持平或略降，增速0-10%，ROE<12% |
| 2-3分 | 订单下降>20%，或核心客户流失，或出现颠覆性竞争者 |
| 0-1分 | 主营业务受根本性冲击，业绩大幅亏损无改善迹象 |

**数据必须来源（优先级）：** 公司财报 > 交易所公告 > 行业协会 > 第三方统计

---

### 维度②：催化剂（权重30%）—— vs 全市场所有股票

> **最重要维度（权重30%）。催化剂与基本面并列为最高权重，决定股价何时启动。**

| 分数 | 核心标准 |
|------|---------|
| 10分 | 1个月内重大催化（合同落地/业绩超预期/政策批准），且股价尚未完全反映 |
| 8-9分 | 1-2个月内重大催化，且催化前股价未完全反映 |
| 6-7分 | 2-3个月内有催化，或短期有数据验证 |
| 4-5分 | 3-6个月有催化，但时间较远 |
| 2-3分 | 仅有"远期想象空间"，无近中期催化 |
| 0-1分 | 已知风险即将兑现（业绩下修/限售解禁/政策利空） |

**数据必须来源：** 公司公告 > 监管批文 > 竞争对手交叉验证 > 分析师研报

---

### 维度③：板块趋势（权重10%）—— vs 全市场所有板块

| 分数 | 核心标准 |
|------|---------|
| 10分 | 板块近20日涨幅全市场前5% + 资金净流入 + 出现政策/数据正面催化 |
| 8-9分 | 板块近20日涨幅全市场前20% + 资金持续流入 + 逻辑清晰 |
| 6-7分 | 板块涨幅持平大盘，无明显资金异动 |
| 4-5分 | 板块涨幅落后大盘，主线资金有明显撤出迹象 |
| 2-3分 | 板块资金连续3周净流出，或出现重大政策利空 |
| 0-1分 | 板块内出现批量暴跌/政策封杀 |

**数据必须来源：** 东方财富板块监测 > Wind > 申万/中信行业涨跌榜 > ETF资金流向

---

### 维度④：资金流向（权重10%）—— vs 个股自身历史

| 分数 | 核心标准 |
|------|---------|
| 10分 | 主力连续3周净流入 + 股东户数下降 + 筹码集中 + 北向加仓 |
| 8-9分 | 主力近1个月净流入 + 量价配合健康 |
| 6-7分 | 主力中性，无明显方向 |
| 4-5分 | 主力近1个月净流出（借利好减仓） |
| 2-3分 | 主力连续3周净流出 + 股东户数增加（筹码分散） |
| 0-1分 | 高位放量滞涨 + 主力大规模出逃 |

**数据必须来源：** 东方财富主力资金流向 > 同花顺筹码 > 港股通持股 > 龙虎榜

---

### 维度⑤：宏观大盘（权重10%）—— vs 历史极端区间

| 分数 | 核心标准 |
|------|---------|
| 10分 | 大盘处于历史极端低估区域，全市场PE<历史20%分位 → 加仓窗口 |
| 8-9分 | 大盘处于合理偏低估区间，政策底出现，流动性宽松 |
| 6-7分 | 大盘估值中性，无极端信号，保持常规仓位 |
| 4-5分 | 大盘估值偏高，全市场PE>历史70%分位 |
| 2-3分 | 大盘极端高估，全市场PE>历史90%分位 |
| 0-1分 | 系统性风险：金融危机/政策黑天鹅/地缘冲击 → 大幅降仓 |

**数据必须来源：** 沪深300 PE分位 > S&P 500 PE分位 > 国家统计局PMI > 央行政策

---

### 维度⑥：股权结构（权重10%）—— vs 同行业水平

| 分数 | 核心标准 |
|------|---------|
| 10分 | 高管/实控人近期增持>3次 + 质押比例<10% + 无大额解禁 |
| 8-9分 | 无减持公告，质押稳定，筹码结构健康 |
| 6-7分 | 有少量质押但无异常，股权结构无明显问题 |
| 4-5分 | 有询价转让但价格接近市价，实控人有少量减持 |
| 2-3分 | 实控人大额询价转让（>3%），或质押比例>30%，或大规模解禁在即 |
| 0-1分 | 高质押比例>50% + 实控人持续减持 + 大量解禁即将到来 |

**数据必须来源：** 交易所公告 > 东方财富F10 > 公司公告回购/增持 > 天眼查

## 六、技术辅助参考（不计入综合分）

> 技术分析**不独立评分**，而是作为催化剂/基本面的**验证信号**，在买入时机选择时提供参考。

| 技术信号 | 对应的维度影响 | 含义 |
|---------|-------------|------|
| 上升趋势 + 缩量回踩均线 | 催化剂 +0.5分 | 提供更好的买入价位 |
| 突破关键阻力位 | 催化剂 +0.5分 | 基本面逻辑被市场验证 |
| 放量滞涨/顶部背离 | 基本面酌情-0.5分 | 警惕股价已透支基本面 |
| 下降趋势 | 催化剂 -1分 | 即使有催化剂，时机不对也不追 |
| RSI>80超买 | 催化剂 -0.5分 | 短线回调风险 |

## 七、数据使用铁律

```
铁律一：L1来源存在时，绝不使用L2-L5替代
铁律二：休市日不得出现任何当日交易价格（标注为"最后收盘价"）
铁律三：存疑数据→标注"无数据，待确认"，绝不在报告中猜测
铁律四：多股扫描时，所有股票必须使用同一数据日期
铁律五：每次报告输出前，必须在开头附【数据时效声明】
```

## 八、研究流程

```
第一步：候选股票池建立（每周扫一次）
  触发条件（任一满足即入池）：
    ✅ 基本面利好信号（订单/业绩/新品）
    ✅ 催化剂出现（政策/公告/数据）
    ✅ 板块出现资金异动
    ✅ 突发消息（24小时内评估）

第二步：对池内所有股票打分
  6个维度统一打分（10分制）
  按综合分高→低排序 → 形成池内相对排名

第三步：信号评估
  买入候选：综合分 ≥ 6.5
  持有：     5.0 ≤ 综合分 < 6.5（逻辑未破坏）
  观望：     4.0 ≤ 综合分 < 5.0
  减仓：     综合分 < 4.0

第四步：操作计划
  8.5分以上：重仓买入方案（价格+仓位+止损）
  6.5-8.4分：正常买入方案
```

## 九、持仓动态管理

```
持仓股出现以下任一情况 → 立即触发重新评估：
  🚨 综合分跌破 5.0（退出信号）
  🚨 基本面核心逻辑破坏（订单丢失/业绩大幅下修）
  🚨 催化剂消失（利好未兑现/变利空）
  🚨 板块资金连续3周净流出

持仓股持续满足以下条件 → 持有不动：
  ✅ 综合分维持在 ≥ 6.5
  ✅ 基本面逻辑未被破坏
  ✅ 目标价未达成
  ✅ 池内有更好的替代机会（排名高出2位以上）→ 考虑换仓
```

## 十、框架使用纪律

```
1. 不为打分而打分 → 目的是发现研究盲点，不是追求高分
2. 分数变化比绝对值重要 → 从7分降到6分要追问原因
3. 分数要每周末更新 → 市场在变，分数也要更新
4. 池子不超过20只 → 超过就主动出池，保持聚焦
5. 出现1分的维度 → 该维度对应风险必须在操作中体现
6. 每次评分必须先搜数据 → 不得凭记忆输出，数据真实性第一
7. 计算后必须用bvvb_calculator.py验算 → 结果才可发出
```
