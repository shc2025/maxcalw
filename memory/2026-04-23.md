## CRSI 指标系统

### 技能位置

- `/workspace/skills/crsi-calculator/__init__.py`

### CRSI 算法（已验证与 TradingView 100% 一致）
```
CRSI = (ta.rsi(close, 3) + ta.rsi(updown_streak, 2) + ta.percentrank(ta.roc(close, 1), 100)) / 3
```

### 关键实现细节
- **percentrank**: `cnt(valid < current) / cnt(valid) * 100`，返回 0-100
- **updown_streak**: 首日=0；相等=0；上涨=(prev≤0?1:prev+1)；下跌=(prev≥0?-1:prev-1)
- **RSI**: Wilder平滑算法（alpha=1/period），与 `ta.rsi` 完全一致
- **percentrank lookback**: 100（100天/周）

### CRSI 信号区间
- <15: 🔴极度超卖（买点信号：上穿15）
- 15-30: 🟠低估
- 30-45: 🟡偏弱
- 45-55: 🟡中性
- 55-70: 🟢偏强
- 70-85: 🟠超买（⚠️谨慎追高）
- >85: 🔴极度超买（⚠️⚠️历史极罕见，约1%时间）
